HTML (от англ. HyperText Markup Language — «язык гипертекстовой разметки») — стандартизированный язык программирования документов во Всемирной паутине. Большинство веб-страниц содержат описание разметки на языке HTML (или XHTML). Язык HTML интерпретируется браузерами; полученный в результате интерпретации формати
